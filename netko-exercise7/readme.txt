1. Knacken Sie das Caesar-Chiffre aus der Datei Caesar.txt. Wie lautet für dieses Chiffre der Schlüssel?
Hinweis: Es ist Ihnen freigestellt, wie Sie das Problem lösen (Brute-force-Angriff, Häufigkeitsanalyse,...) Sie können sich gerne ein Java-Programm schreiben oder das Internet zu Rate ziehen. Wichtig ist nur, dass Sie Ihr Vorgehen verstanden haben.
2. Lösen Sie das Vigenère-Chiffre aus der Datei Vigenere.txt. Wie lautet für dieses Chiffre der Schlüssel?
Hinweis: Wenn Sie die Schlüssellänge (n) kennen, ist die Dechiffrierung so, als ob Sie n monoalphabetische Einzelchiffren hätten. Für die Ermittlung der Schlüssellänge stehen ihnen der Kasiskitest oder Friedmanntest zur Verfügung (Diese beiden Tests müssen Sie nicht zwangsläufig verstanden haben)
3. Schreiben Sie ein Java-Programm, das den Text aus der Datei AES.txt mithilfe des Verschlüsselungsverfahrens AES verschlüsselt.
Hinweis: Für Verschlüsselungen und Hash-Bildung steht Ihnen in Java das Package "java.security" zur Verfügung.
4. Vergleichen Sie die Verschlüsselungsgeschwindigkeiten von AES und DES anhand der Beispieldatei "GrosseDatei.dat". Wie lange brauch AES zum Verschlüsseln und wie lange benötigt DES für diese Vorgang?

Folgende Dinge sollten Sie abgeben:
- Klartext des Caesar-Chiffres (Datei)
- Schlüssel des Caesar-Chiffres (Text)
- Klartext des Vigenere-Chiffres (Datei)
- Schlüssel des Vigenere-Chiffres (Text)
- Mit AES verschlüsselter Chiffretext (Datei)
- Java-Programm zur AES-Verschlüsselung (Datei)
- Vergleich der Verschlüsselungszeiten von AES und DES (Text)
- ggf. Java-Programm zur DES-Verschlüsselung (Datei)
